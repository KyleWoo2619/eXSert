# eXSert

A 3D action combat game set aboard a derelict spaceship. Fight through waves of enemies using combo attacks, stance switching, and aerial maneuvers.


===============================================
HOW TO PLAY
===============================================

INSTALLATION
1. Download the game build
2. Extract the ZIP file
3. Run eXSert.exe


CONTROLS

Keyboard + Mouse:
- WASD - Move
- Mouse - Look around
- Left Click / X - Light Attack
- Right Click / Y - Heavy Attack
- Space - Jump
- Shift - Dash
- LB / Q - Change Stance
- Guard - Block (Hold)
- ESC - Pause Menu

Controller (Xbox/PlayStation):
- Left Stick - Move
- Right Stick - Look around
- A / X - Jump
- X / Square - Light Attack
- Y / Triangle - Heavy Attack
- B / Circle - Dash
- LB / L1 - Change Stance
- RB / R1 - Guard
- Start - Pause Menu

- Guard Timing: Perfect guards can create openings
